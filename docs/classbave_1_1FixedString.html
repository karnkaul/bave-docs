Detailed Description

template<std::size_t Capacity = 64>
class bave::FixedString< Capacity >

Simple "stringy" wrapper around a stack buffer of chars.

Stores a buffer of Capacity + 1 to guarantee null termination. Note: chars in input strings beyond Capacity are silently dropped.
